SEARCHING FOR CONNECTION
Format: 11x17" risograph poster series / 2020 — present

This project uses the wi-fi network information as inspiration for poster designs. Placed in their owner's homes, each work hides these sets of information in plain sight, functioning as network card. No longer will a user rummage for that sad piece of cardstock, buried in a random drawer or become lost in the laundry— rather, the information is easily accessible and hard to misplace.
