BLOX TYPEFACE
Format: 1920x1080 video / 1 minute / 2020

Blox is a moving variable typeface. Letters were created using a combination of analog and digital processes. Iterations of each letter were altered with an image rasterizer, run through Processing, to render individual frames. The work was primarily inspired by Roblox characters found on Sketchup, a community-sourced library of 3D rendered objects.
