INTERGALACTIC DRIFT
Format: Website / HTML, CSS, Javascript, jQuery / 2019

This data visualization project studies the expanding nature of dark matter in space by mapping the distance between galaxies. Distance is mapped in lightyears and has been translated to pixels to provide accurate measurements. Screen size expands infinitely— if accessible, this work is best navigatee with a touch pad.
